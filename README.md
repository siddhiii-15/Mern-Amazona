# 🛒 MERN Amazona

A full-stack e-commerce web application built using the **MERN Stack (MongoDB, Express.js, React.js, and Node.js)**. The application demonstrates a complete online shopping workflow, including authentication, product browsing, shopping cart management, order processing, and an administrative dashboard.

This repository is maintained as a learning and portfolio project and will continue to evolve with new features, UI/UX improvements, and performance optimizations.

---

## Current Features

- User Authentication (JWT)

## Installation

### Clone the repository

```bash
git clone https://github.com/siddhiii-15/Mern-Amazona.git
```

### Install Backend Dependencies

```bash
cd backend
npm install
```

### Install Frontend Dependencies

```bash
cd frontend
npm install
```

### Configure Environment Variables

Create a `.env` file inside the **backend** directory and configure the required environment variables.

Example:

```
MONGODB_URI=your_database_connection
JWT_SECRET=your_secret_key
PAYPAL_CLIENT_ID=your_paypal_client_id
GOOGLE_API_KEY=your_google_maps_api_key
```

### Run Backend

```bash
npm start
```

### Run Frontend

```bash
npm start
```
